== Description ==
If you are looking for a WooCommerce Table Rate Shipping plugin then look no further! A super easy to use plugin that will provide your WooCommerce site with table rate shipping.

It is super-easy to add to your WooCommerce store and set up shipping rules.

= Key Features =
* Unlimited countries/shipping zones
* Super easy to use - very intuiative user interface
* You can tier your rates by weight or cart-totals - effective for most stores
* Add additional handling fees

== Installation ==


1. Upload `woocommerce-easy-table-rate-shipping.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Woocomerce-> Settings-> Shipping-> Table, in your Wordpress site
4. Make sure the 'Enabled' checkbox is ticked.
5. Add your shipping rates and you are all set!

== Frequently Asked Questions ==

= How do I specify no upper limit on the max rate? =
Simply use the '*' (asterisk) in the 'Max' field and the plugin will interpret that as infinite.

= What countries & currencies are supported? =
We support ALL WooCommerce countries and currencies. If it is in WooCommerce, then we have it covered!
